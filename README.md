# AI Research Assistant (RAG System)

A Retrieval-Augmented Generation (RAG) system built using:

- LangChain
- FAISS
- HuggingFace Embeddings
- Google Gemini
- Streamlit

## Features

- PDF Document Loading
- Text Chunking
- Semantic Search
- FAISS Vector Database
- Gemini-Powered Answers
- Streamlit Web Interface

## Project Flow

PDF → Chunking → Embeddings → FAISS → Retrieval → Gemini → Answer
